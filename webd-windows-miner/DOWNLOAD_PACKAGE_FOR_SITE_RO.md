# Pachet download pentru site - WebDollar Windows Miner (Beta 0.0.3)

Data: 2026-04-10

Acest fisier contine lista fisierelor pe care le poti publica pe site pentru download.

## 1) Kit instalare aplicatie

1. Installer Windows (recomandat)
- Fisier: `dist/WebDollar Windows Miner Setup 0.0.3.exe`
- Descriere pentru site: "Installer Windows (x64) - instalare standard"

2. Varianta portabila (fara instalare)
- Fisier: `dist/win-unpacked/WebDollar Windows Miner.exe`
- Descriere pentru site: "Executabil portabil Windows (x64)"

3. Blockmap (optional pentru update-uri diferentiale)
- Fisier: `dist/WebDollar Windows Miner Setup 0.0.3.exe.blockmap`
- Descriere pentru site: "Fisier tehnic pentru update incremental (optional)"

## 2) Documentatie pentru download

1. Ghid complet instalare + utilizare
- Fisier: `USER_GUIDE_BETA_RO.md`
- Descriere pentru site: "Ghid complet (RO)"

2. Ghid complet instalare + utilizare (EN)
- Fisier: `USER_GUIDE_BETA_EN.md`
- Descriere pentru site: "Full guide (EN)"

3. Ghid foarte scurt pentru non-tehnic
- Fisier: `USER_GUIDE_BETA_RO_SHORT.md`
- Descriere pentru site: "Ghid rapid (RO) - 1 minut"

4. Ghid foarte scurt pentru non-tehnic (EN)
- Fisier: `USER_GUIDE_BETA_SHORT_EN.md`
- Descriere pentru site: "Quick guide (EN) - 1 minute"

5. Inventar API Node-WebDollar (optional pentru developeri)
- Fisier: `NODE_WEBDOLLAR_API_INVENTORY_RO.md`
- Descriere pentru site: "Referinta API (RO)"

## 3) Structura recomandata pe pagina de download

- Sectiune "Aplicatie":
  - Installer Windows (.exe)
  - Executabil portabil (.exe)
- Sectiune "Documentatie":
  - Ghid complet
  - Ghid complet EN
  - Ghid rapid non-tehnic
  - Ghid rapid EN
- Sectiune "Developer resources" (optional):
  - Inventar API Node-WebDollar

## 4) Nota pentru publicare

- Pentru majoritatea utilizatorilor, pune primul buton pe installer:
  - `WebDollar Windows Miner Setup 0.0.3.exe`
- Varianta portabila sa fie al doilea buton (advanced users).
