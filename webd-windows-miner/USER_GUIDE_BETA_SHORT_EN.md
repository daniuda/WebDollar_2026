# WEBDOLLAR Windows Miner - Very short guide (non-technical)

Version: 0.0.3

## 1) Install
1. Open: WebDollar Windows Miner Setup 0.0.3.exe
2. Click Next until finish.
3. Open app from Start Menu or Desktop.

## 2) Quick setup
1. Fill Pool API URL.
2. Fill Wallet address.
3. Click Save config.

## 3) Wallet
- No wallet yet: click Generate wallet.
- Existing wallet: click Import .webd file.
- If wallet is encrypted: enter password and click Unlock.

## 4) Mining
- Start mining = start.
- Stop mining = stop.

## 5) Send WEBD
1. Enter recipient address.
2. Enter amount.
3. Click Send WEBD.
4. Confirm in dialog (YES/NO).

If you choose NO, transfer is canceled.

## 6) If transfer is not visible immediately
- This can be normal.
- App keeps checking automatically for longer.
- Use the transaction check link shown in the app.

## 7) If something fails
1. Close app fully.
2. Start app again.
3. Verify pool URL and wallet address.
4. Retry.
