# WEBDOLLAR Windows Miner - Ghid foarte scurt (non-tehnic)

Versiune: 0.0.3

## 1) Instalare
1. Deschide fisierul: WebDollar Windows Miner Setup 0.0.3.exe
2. Apasa Next pana la final.
3. Deschide aplicatia din Start Menu sau Desktop.

## 2) Setare rapida
1. Completeaza Pool API URL.
2. Completeaza Wallet address.
3. Apasa Save config.

## 3) Wallet
- Daca nu ai wallet: apasa Generate wallet.
- Daca ai wallet: apasa Import .webd file.
- Daca wallet-ul este salvat criptat: introdu parola si apasa Unlock.

## 4) Mining
- Start mining = porneste.
- Stop mining = opreste.

## 5) Trimitere WEBD
1. Scrie adresa destinatarului.
2. Scrie suma.
3. Apasa Trimite WEBD.
4. Confirma mesajul (DA/NU).

Daca alegi NU, transferul nu se face.

## 6) Daca transferul nu apare imediat
- Este normal sa dureze.
- Aplicatia verifica automat mai mult timp.
- Verifica apoi tranzactia din link-ul de verificare afisat in aplicatie.

## 7) Daca ceva nu merge
1. Inchide aplicatia complet.
2. Deschide din nou aplicatia.
3. Verifica URL-ul de pool si adresa wallet.
4. Reincearca.
